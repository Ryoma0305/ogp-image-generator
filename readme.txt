Generate OGP image automatically when a post save.

== Description ==


* You can generate OGP image automatically when a post save.

* how to generate a OGP image
   * Fill in the entry form items on the settings screen.
   * Edit and save the article for which you want to create an OGP image. The article title will then be inserted and the OGP image will be automatically generated in the destination folder.

== Installation ==

1. From the WP admin panel, click “Plugins” -> “Add new”.
2. In the browser input box, type “OGP Image Generator”.
3. Select the “OGP Image Generator” plugin and click “Install”.
4. Activate the plugin.
